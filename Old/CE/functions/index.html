基本函数
函数，是一个高级编程语言中十分重要的东西,为此C!提供了较多的函数，0.8.1.6的版本的c!还提供了dll加载与调用 (*0) 来提供对扩展的支持.

SupportFunctions,_SF
该函数可以查看目前支持的所有函数与关键字，函数返回函数与关键字的总个数。
例子：
var in = _SF();
println(in);//println在0.8.1.6加入

输出:
SupportFunctions: |arguments
.....//省略没必要的东西
52

print,println
该函数可以实现输出，仅支持一个参数，参数可用"+"号拼接。其中print不自动换行。
例子：
var a = "Hello",b = "World";
println(a+b+"!");//println在0.8.1.6加入

输出:
HelloWorld!

input,CE$input
该函数可以实现输入，最多支持一个参数，最少可没有参数，参数实质上为对变量的引用。返回输入的值，其中CE$input为不显示输入，适合输入密码！但是，CE$input有些问题，所以暂时不建议使用！
例子：
var a,b,c,d; a = input(); input(b); c = CE$input(); CE$input(d); print(a+b+c+d+"\n");

输入:
ILoveC
ILoveJava
ILoveC++
ILoveC#
(*1)
输出:
ILoveCILoveJavaILoveC++ILoveC#

math
该函数可以计算加减乘除，注意无法计算sin等函数，只可以加减乘除（目前）！！！，可以使用'$'开头来引用程序的变量并转换为数字。和它类似的还有关键字cmath
例子：
var a = "100",b = "2";
println(math("$a/$b"));
b = 0;
println(math("$a/$b"));

输出:
50.000000
inf
(*2)

runProgram,system
该函数可以实现命令行指令。
例子：
system("start cmd");//相当于 runProgram("start cmd");

输出:
//没有输出,弹出一个cmd窗口

local
该函数可以输出变量的所有（允许展现）信息（该信息为程序处理变量时的真正的内部信息），但是变量属性必须带有OUTABLE,否则默认没有权限，可以使用#pragma CheckApi false来禁用检查。
例子：
CESTDAPI OUTABLE int out0 = 0;
local(out0);

输出:
variable name :out0
variable value :0
variable virtual localtion :0x0014
variable's attributes :
const :false
inorecase :false
depth :0

rand
该函数可以实现随机数，仅支持三个参数。第一个为最小值，第二个为最大值，第三个为存储变量的引用，返回生成的随机值。注意：最大值与最小值不可以相等，否则解释程序会无故退出！
例子：
var a,b;
rand(0,100,a);
println(a);
b = rand(0,100,a);
println(a+" "+b);

输出(这只是一个可能的版本):
60 34 34

declare_var
这个函数将会在未来与XXXX_var一起推出，目前bug较多，勉强可以使用，与API:to_real定义变量类似。

SetConsolePosition,CE$setPos
该函数可以设置控制台光标的位置。这里不方便演示，第一个参数为列数，第二个为行数。

print$delay,print$d
该函数可以实现每个字符延时输出，演示！第一个参数为时间，第二个为字符串！
例子：
print$d(100,"HelloWorld!");

print$color,print$c
该函数可以实现输出多彩的字符串，第一个参数为颜色(*3)，第二个为字符串！
例子：
print$c(CL_YELLOW,"HelloWorld!");

输出:
HelloWorld!

clear
该函数可以实现清除屏幕，相当于system("cls");或者runProgram("cls");
例子：
print("123");
Sleep(1000);//方便演示
clear();
Sleep(1000); print("456");

输出(点我来演示):
//等待演示

ceEdit,cpowershell
该函数可以让你进入内置的简单运行器.无法演示，没有参数！

free
该函数可以支持释放变量（包括系统变量（以后会修复的））（就是从变量表中清除，变量名必须是个字符串）。
例子：
BaseObj a = 10;
seek();
free("a");
seek();

输出:
variable's name! value!
....省略非非关键字段
author__
a 10
variable's name! value!
....省略非非关键字段
author__

myFriend
这是一个彩蛋函数，接受一个字符串，自己试试吧。
例子：
myFriend("森下下士");

输出:

time
返回目前时间的字符串形式，必须接受一个引用，并把字符串的时间存储在引用中
例子：
var a;
time(a);
print(a);

输出:
Fri Aug 27 13:27:43 2021

getKeyInt,getKeyChar
利用c/c++的conio.h中的_getch获取的值并返回，getKeyInt返回数字，getKeyChar返回字符，都只需要一个引用作为参数
例子：
var a;
getKeyInt(a);
//输入了'q' println(a);
getKeyChar(a);
//输入了'q' println(a);

输出:
//输入 q 113 //输入 q q

getMousePosX,getMousePosY
利用c/c++的conio.h中的方法获取相对于整个屏幕的鼠标坐标，需要一个引用，返回NULL
例子：
var a;
getMousePosX(a);
//输入了'q' println(a);

输出:
//一种可能 792

getThisMousePosX,getThisMousePosY
利用c/c++的conio.h + windows API中的方法获取相对于控制台窗口的鼠标坐标，需要一个引用，返回NULL,(注：该函数需要运行在WinXP以上，否则不支持)
例子：
var a;
getThisMousePosX(a);
//输入了'q' println(a);

输出:
//一种可能 120

高级函数
此章节讲述了高级的ce编程

LoadInstDLL,LoadDLL
载入dll,接受两个参数，第一个参数为dll加载后的别名，第二个为dll的地址,返回第一个参数的值,当然如果嫌麻烦，可以吧dll的地址当做dll的别名.
例子：
var a = "myDLL",dllPath = "d0.dll";
LoadDLL(a,dllPath);
seek_dll();

输出:
myDLL 0x69480000

UnloadInstDLL,UnloadDLL
卸载dll,接受一个参数，第一个参数为dll加载后的别名,返回NULL.
例子：
var a = "myDLL",dllPath = "d0.dll";
LoadDLL(a,dllPath);
UnloadDLL(a);

（暂无输出）

CalInstDLL,CalDLL
调用dll中的方法，接受两个参数，第一个为dll的别名，第二个为函数的名字，加载的函数必须不接受或者不使用参数，也不返回任何值，c/c++中格式如下: void NAME(void){...},调用中出现异常会直接崩溃！
例子：
var a = "myDLL",dllPath = "d0.dll";//悬停在这里可查看d0.dll部分源代码(c++)
LoadDLL(a,dllPath);
CalDLL(a,"A");
UnloadDLL(a);

输出:
HelloWorld!

seek_dll
查看所有dll的列表
见LoadInstDLL（点我跳转至例子）

*0:目前的版本仅支持C/C++ DLL的无返回值无参数的函数,如void a(void){...}, 例如这样的: int a(int);不支持，会强制转成void (*)(void)类型。

*1:其中，IloveC++与ILoveC#不会显示出来！

*2:C!一直很宽容，允许程序猿做许多擦语法球的事（实际上是我懒得优化了），因此100/0不会报错，返回inf

*3:color可以使用c! installer的sysVars.rb中的CL_XXX为参数，它定义了十分多的颜色可以选择，但是切记，不可以在命令行参数为ce -l xxxx,或者ce -e中输入CL_XXX，因为这两个参数c!是不会加载系统变量的（方便给你最干净的环境与最快的加载速度（其实是我懒得做了））。
